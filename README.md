# VLA Test Planning

VLA 模型测试应用规划仓库 —— 管理应用分类体系、组件覆盖分析、测试应用池构建。

## 两条路径

| 路径 | 文件 | 定位 | 应用数 | 开发方式 | 验收方式 |
|------|------|------|--------|---------|---------|
| pm-cockpit 精细版 | `app-pool-plan.md` | PRD 驱动，精确控制 | 16 个 | PRD + Fabrice 开发 | Moss + mano-cua |
| mano-afk 端到端版 | `app-pool-plan-v4.md` | 一句话 prompt，自动化 | 13 个 | mano-afk 自动开发 | mano-cua 自动验收 |

**区别：**
- pm-cockpit 版追求 100% 组件覆盖 + 精确边界控制，含 mano-afk ❌ 的复杂应用
- mano-afk 版只保留双适配（afk 能做 + cua 能验）的应用，覆盖 65/69 组件（94%）
